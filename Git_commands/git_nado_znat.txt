Для демонстрации необходимо уметь:
1.	Клонировать проект; 						+ git clone
2.	Инициализировать новый репозиторий; 				+ git init
3.	Добавлять в индексацию файлы и фиксировать изменения. 		+ git status, add
4.	Создавать новые ветки; 						+ git branch, checkout
5.	Переключаться между ветками; 					+ git checkout
6.	Обновлять локальную копию удалённого репозитория;   		+ git pull
7.	Переносить изменения из удалённого репозитория в локальный; 	+ git fetch (git push - наоборот)
8.	Работать с файлом .gitignore; 					+
9.	Использовать инструменты поиска и отладки;			+
10.	Производить слияние веток;					+ git merge 
11.	Разрешать конфликты;    					+ git merge --continue
12.	Прятать изменения;						+ git stash 
13.	Работать с указателями;						+ 
14.	Производить отмену изменений разными способами;			+ git restore, reset 
15.	Переключаться на другой коммит;					+ git checkout
16.	Восстанавливать изменения на момент указанного коммита;		+ git checkout
17.	Копирование и перенос коммитов;					+ git log, cherry-pick 
18.	Сжатие коммитов;						+ git rebase -i HEAD~2 (squash)
